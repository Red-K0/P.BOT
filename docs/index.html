Bot
Contains methods for initializing and preparing the bot's client.

Bot.Secrets
Allows the fetching of private data from user secrets.

Bot.Client
P.BOT's main client.

Bot.ClientH
P.BOT's HTTP client, used for external network requests.

Bot.GuildID
The current guild's ID, fetched via Bot.GetSecret(string).

Bot.GuildURL
The base guild URL.

Bot.GetSecret(string)
Extracts a value from the project secrets, returning null if the identifier does not exist.

Bot.Restart
Stops the client, releases its resources, and restarts the bot client.

Bot.Start()
Starts the bot's client, initializes event handlers, and prepares caches.

Caches.Messages
Contains the recent messages cache and its relevant methods.

Caches.Messages.Recent
Contains all messages in the bot's cache.

Caches.Members
Contains the cached members list, role list, and their relevant methods. This class possesses a static constructor not shipped in the GitHub repo for privacy reasons.

    Embeds
    Contains functions for the creation of embeds and their components.

    Embeds.CreateAuthor(string, string, string)
    Creates an EmbedAuthorProperties object for use in an Embed.
    Parameters:
  • authorText:The string to display, used for the EmbedAuthorProperties.Name property.
  • iconURL:The string containing the URL of the image to display to the left of the text, used for the EmbedAuthorProperties.IconUrl property.
  • textURL:The string containing the URL to set authorText as a hyperlink towards, used for the EmbedAuthorProperties.Url property.

    • Embeds.CreateFooter(string, string)
    Creates an EmbedAuthorProperties object for use in an Embed.
    Parameters:
  • footerText:The string to display, used for the EmbedFooterProperties.Text property.
  • iconURL:The string containing the URL of the image to display to the left of the text, used for the EmbedFooterProperties.IconUrl property.

    • Embeds.CreateField(string, string, bool)
    Creates an EmbedFieldProperties object for use in an Embed.
    Parameters:
  • name:The string to display at the field title, used for the EmbedFieldProperties.Name property.
  • value:The string to display in the field, used for the EmbedFieldProperties.Value property.
  • noInline:Whether or not to display the field inline.

    • Embeds.GetAssetURL(string)
    Gets the GitHub hosted URL of the given asset.
    Parameters:
  • fileName:The name of the hosted file.

    • Embeds.ASSETS
    The URL to the GitHub Assets folder.

    Embeds.STD_COLOR
    The hex code for standard embed gray.

    Embeds.RandomColor
    Generates a random 24-bit integer from the current time in ticks.

    Embeds.Generate(EmbedProperties, string[], string, bool)
    Combines the specified parameters to generate an embed.

    Embeds.ToEmbed(RestMessage, string, EmbedFooterProperties, string, string)
    Combines the specified parameters with parameters extracted from a given RestMessage to generate an embed.
    Parameters:
  • message:The RestMessage object to extract parameters from.
  • url:A link to an address of a webpage. When set, the title becomes a clickable link, directing to the URL. Additionally, embeds of the same URL are grouped.
  • footer:Contains the EmbedFooterProperties to be used in the embed.
  • title:The text that is placed above the description, usually highlighted. Also directs to a URL if one is given in url, has a 256 character limit.
  • thumbnail:The URL of thumbnail of the embed, displayed as a medium-sized image in the top right corner of the embed.

    • Embeds.Title(string, EmbedFieldProperties[], ulong)
    Creates a MessageProperties object suitable for Commands.SlashCommands.GetTitle(Commands.Helpers.Library.Titles).
