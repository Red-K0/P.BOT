Caches.Messages.Recent
Contains the most recent messages in the bot's cache.

Caches.Messages.Get(ulong)
Iterates through the Caches.Messages.Recent list in reverse order, returning the Message corresponding to the given ID if it's found, returning null otherwise.

Caches.Messages.Add(Message)
Adds the given Message to the Caches.Messages.Recent list, removing the oldest message if the list size is at Caches.Messages.RECENT_CACHE_SIZE_MAX.

Caches.Messages.Edit(Message)
Updates a message in the cache if it's found, otherwise creating it as a new entry.

    The hex code for standard embed gray.

    Embeds.Generate(string, EmbedAuthorProperties, Nullable{DateTimeOffset}, EmbedFooterProperties, int, ulong, string[], string, string, string, bool, EmbedFieldProperties[], ulong)
    Combines the specified parameters to generate an embed.
    Parameters:
  • description: The section of the embed where the main text is contained, limited to 4096 characters.
  • authorObject: Contains the EmbedAuthorProperties to be used in the embed.
  • timestamp: Displays time in a format similar to a message timestamp. Located next to the footerObject.
  • footerObject: Contains the EmbedFooterProperties to be used in the embed.
  • RGB: The hex code indicating an embed's accent color. Defaults to a random value when not set or -1.
  • replyTo: The ID of the message being replied to with the embed.
  • imageURLs: An array of URIs for the attachments included in the embed, displayed below the description element.
  • thumbnailURL: The URL of thumbnail of the embed, displayed as a medium-sized image in the top right corner of the embed.
  • title: The text that is placed above the description, usually highlighted. Also directs to a URL if one is given in titleURL, has a 256 character limit.
  • titleURL: A link to an address of a webpage. When set, the title becomes a clickable link, directing to the URL. Additionally, embeds of the same URL are grouped.
  • ephemral: Creates an ephemeral message when set to true.
  • fieldObjects: Contains an array of EmbedFields to include in the embed
  • refID: The ID of the user responsible for the embed's creation.

    • Embeds.Generate(RestMessage, string, EmbedFooterProperties, Nullable{ulong}, string, string)
    Combines the specified parameters with parameters extracted from a given RestMessage to generate an embed.
    Parameters:
  • targetMessage: The RestMessage object to extract parameters from.
  • titleURL: A link to an address of a webpage. When set, the title becomes a clickable link, directing to the URL. Additionally, embeds of the same URL are grouped.
  • footerObject: Contains the EmbedFooterProperties to be used in the embed.
  • replyTo: The ID of the message being replied to with the embed.
  • title: The text that is placed above the description, usually highlighted. Also directs to a URL if one is given in titleURL, has a 256 character limit.
  • thumbnailURL: The URL of thumbnail of the embed, displayed as a medium-sized image in the top right corner of the embed.
